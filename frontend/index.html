数据源

JavBus / JavDB 为主；AVSOX 专收无码、AVMOO 收录大量旧番号；FC2 专收 FC2-PPV 素人/无码（fc2ppvdb.com，须配 FlareSolverr）

JavDB 反爬（1.4.2）

留空即自动探测本机/同宿主机已部署的 FlareSolverr（含扫描 docker 同网段），装了就自然连上、零填写；探不到则回退增强直连，且一段时间内不再重扫。也可手填具体 URL（局域网 IP 或外网域名）固定使用。后端已做串行+排队上限+熔断，多任务并发也不会把它压垮
默认开。若 FlareSolverr 报 ERR_NO_SUPPORTED_PROXIES，或它自带网络出口（如 WARP），请关掉走直连
可选。浏览器登录 JavDB 后导出的 Cookie，用于过盾/提升抓取成功率
默认关：点开「JavBus+JavDB」详情时才补抓样品图（省资源）。开启后翻页就后台提前抓好，点开即时显示，但 FlareSolverr 请求更多、更耗资源
检测是否可达 / 是否被 CF 拦截 / 出口 IP 所在国（判断是否需要日本 IP）

FC2 数据源（1.4.3）

fc2ppvdb.com 强制 Cloudflare Turnstile 人机验证，必须经 FlareSolverr 才能抓取。留空时：先复用上方 JavDB 填的地址，JavDB 也留空则自动探测本机/同宿主机的 FlareSolverr
默认开。若 FlareSolverr 自带网络出口或报代理错误，请关掉走直连
可选。浏览器过盾后导出的 fc2ppvdb Cookie，用于提升抓取成功率
默认开。fc2ppvdb 对下架条目常无封面/标题，开启后从 MissAV 补全（封面走 fourhoi CDN）。强烈建议保持开启
MissAV 域名常变。可填多个镜像（逗号分隔）做兜底，如 https://missav.ws,https://missav123.com
默认开，强烈建议保持。fc2ppvdb 的最新列表封顶、够不到市面最新号；sukebei 种子站按 id 倒序＝最新、直连不过盾、最快，能拿到更新的番号。开启后首页最新优先走 sukebei，够量就跳过较慢的 fc2ppvdb 首页；卡片标题/封面较朴素（种子标题 + fourhoi 封面），点开详情时再由 fc2ppvdb/MissAV 补全完整信息。下载仍按番号走 Jackett/sukebei
默认开、低负担。开启后程序在后台串行+节流+直连 MissAV（不过盾）慢慢抓最新片的真标题/封面/样品图灌进缓存：刷新后列表卡标题/封面变干净、点开详情样品图秒出。女优/标签仍点开才走 fc2ppvdb（按你的选择不预抓，避免 FlareSolverr 负担）
后台预抓最新多少条（0＝关闭）。直连 MissAV、每条间隔 0.5 秒，20 条约十几秒抓完且只抓一次（已缓存不重抓），日常几乎无感。最多 60
仅当上面关掉 sukebei、或 sukebei 数量不足时，才回退抓 fc2ppvdb 首页补充。实测其首页不支持翻页（page=2=page=1）、忽略 per_page，一次约 100 条混排，本程序按番号降序取最新 N 条，故默认 1 页即可，多设无益反而每页多过一次盾变慢
检测 fc2ppvdb 是否可达 / 是否被 Turnstile 拦截 / FlareSolverr 是否生效
提示：FC2 站点不提供磁力，下载仍用 Jackett/sukebei 按番号「FC2-PPV-xxxxxxx」检索

抓取与展示

每个数据源最多抓取条目数，最高 500。仅抓列表页速度快，详情翻页时按需加载
未搜索时，首页自动展示所选数据源的最新片源
AVSOX 详情页较慢，数量不宜过大；演员/标签会在翻到该页时渐进加载
